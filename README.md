# 🌱 Agro Forte - Futuro Sustentável

## Equilíbrio entre Produção e Meio Ambiente

Este projeto é um site desenvolvido para conscientizar sobre a importância de unir o desenvolvimento do agronegócio à preservação ambiental.

O objetivo é mostrar que é possível aumentar a produção de alimentos utilizando tecnologias, práticas sustentáveis e respeito aos recursos naturais, garantindo qualidade de vida para as futuras gerações.

## ✨ Funcionalidades

- Página inicial moderna
- Design responsivo (computador e celular)
- Informações sobre agricultura sustentável
- Seção sobre tecnologia no campo
- Destaque para preservação ambiental
- Animações com JavaScript

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🚀 Como executar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.

Ou publique o projeto utilizando o **GitHub Pages**.

## 🎯 Objetivo

Demonstrar que o agronegócio e a preservação ambiental podem caminhar juntos por meio da inovação, da responsabilidade e da sustentabilidade.
